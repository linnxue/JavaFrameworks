#  什么是spring

- spring是一个框架
- 核心思想是IoC，AOP
- 主要目的是简化企业开发，帮助分离对象之间的依赖关系
  - 见前部分代码
- 核心概念
  - **IoC：Inversion of Control 控制反转**
  - **DI：Dependency injection 依赖注入**
  - **AOP：Aspect Oriented Programming 面向切面编程**

# Spring能解决的问题

- 解耦应用程序，简化开发流程
- AOP支持
- 声明式事务
- 集成junit4
- 方便集成各种开源的优秀框架
- 降低 JavaEE API 的使用难度
